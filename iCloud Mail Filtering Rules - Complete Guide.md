HOW TO CREATE RULES IN iCLOUD MAIL:

1\. Open iCloud Mail in web browser  
2\. Click Settings (gear icon) \> Rules  
3\. Click "+" to add new rule  
4\. Enter Rule Name  
5\. Set Condition ("is from" for sender emails)  
6\. Enter email address or domain  
7\. Set Action to "Move to Folder"  
8\. Select destination folder  
9\. Click "Add" to save

ALTERNATIVE: Use Mac Mail App  
The Mac Mail app provides more reliable rule creation:  
1\. Open Mail app  
2\. Go to Mail \> Settings \> Rules  
3\. Create rules (they sync to iCloud automatically)  
4\. More options available (multiple conditions, AND/OR logic)
